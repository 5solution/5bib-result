🌱 5BIB Content Seed Tool

Tự động tạo 5 categories + 5 bài viết hướng dẫn về 5BIB Result + publish ngay. Idempotent — chạy nhiều lần OK, slug đã tồn tại sẽ skip.

Yêu cầu: bạn đang đăng nhập admin (cookie session active). Script sẽ gọi /api/admin/article-categories + /api/admin/articles qua proxy của admin → backend tự động inject Logto Bearer token.

📋 Tiến trình

✅ Tiếp theo

  1. Mở /articles để xem 5 bài viết đã publish
  2. Mở /article-categories xem 5 danh mục
  3. Vào /api-keys tạo key mới name content-web local dev
  4. Copy full key → paste vào content-web/.env.local dòng ARTICLES_API_KEY=ak_xxx
  5. Restart preview content-web (báo Claude restart giúp)
  6. Mở http://localhost:3015 → thấy categories + featured + grid bài viết
